## Requirements

```
1. Java 11

2. Maven 3.x.x
``````

## Setup

**1. Git Clone**

```
git clone https://github.com/lyes-s/java-design-patterns.git
``````

**2. Maven Build**

```
cd java-design-patterns/

mvn clean install
``````
